# Year 8 Unit 5 — Interactive Programming Revision

A student-focused revision experience covering algorithms, Python turtle, sequence, iteration, selection, lists, subprograms, strings and debugging.

## Learning features

- topic exploration and mastery tracking
- live turtle regular-shape generator with copyable Python
- three interactive debugging challenges
- random-choice and string-building experiments
- vocabulary flashcards
- 10-question assessment with instant feedback
- device-local progress and XP
- responsive layout, dark mode and keyboard support

## Live site

https://kaan84k.github.io/unit5-revision/

The project is a dependency-free single HTML file and is deployed through GitHub Pages.
